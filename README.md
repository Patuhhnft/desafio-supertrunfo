# Desafio Super Trunfo 🃏

Repositório de projetos da faculdade.

## Sobre o Projeto

Implementação do clássico jogo **Super Trunfo** em linguagem C, desenvolvido como atividade acadêmica.

O Super Trunfo é um jogo de cartas onde cada carta representa um país com diferentes atributos. Os jogadores comparam os atributos de suas cartas e quem tiver o maior valor em um determinado atributo vence a rodada e fica com a carta do adversário. O objetivo é colecionar todas as cartas.

## Estrutura do Repositório

```
desafio-supertrunfo/
├── README.md
└── supertrunfo.c   # Implementação principal do jogo
```

## Como Compilar e Executar

Você precisa do compilador **GCC** instalado.

```bash
# Compilar
gcc supertrunfo.c -o supertrunfo

# Executar
./supertrunfo
```

## Funcionalidades

- Cadastro de cartas com atributos de países (população, área, PIB, pontos turísticos e densidade demográfica)
- Comparação de cartas entre jogadores
- Exibição das cartas cadastradas

## Tecnologias

- Linguagem C
- Compilador GCC
